## 🚀 What Is CodexDaemon?
**Current version:** `v0.4.0` – Phase 3 (Mutation Risk Engine + Neurogenesis feed).

> “Each scan a signal. Each mutation a whisper. Each commit a neural shockwave.”

CodexDaemon is a **self-inspecting, self-describing AI maintainer** wired directly into GitHub Actions:

- 🧬 **Mutation Engine (GPT-5)** – scores, narrates, and (eventually) edits its own code.
- 🛰️ **Neurogenesis Feed** – mirrors live mutation + risk telemetry into  
  [`codexdaemon-feed`](https://github.com/roninazure/codexdaemon-feed) for public viewing.
- 🧲 **Anti-Gravity Reflex Stabilizer (Gemini-3 Alpha)** – planned drift dampener for hallucination loops.
- 🧠 **Agent Mesh** – designed to route work across GPT-5, Claude, Gemini, and future models.
- 🕸️ **Mad Scientist Network** – wired into Project D.A.R.C., Mad Scientist README engine, and beyond.

This repo is the **“brain”**. The `codexdaemon-feed` repo is its **public pulse monitor**.

- `.github/workflows/codexdaemon-daily-mutation.yml`  
  – Runs CodexDaemon’s mutation + diagnostics pipeline and commits into this README.

  – Injects mutation score block and syncs `README.md` into [`codexdaemon-feed`](https://github.com/roninazure/codexdaemon-feed).
- `.github/scripts/*.py`  
  – Scanners, injectors, and formatters that write into the comment-guarded blocks in this file.

  – Historical logs of CodexDaemon’s behavior and scan output.
